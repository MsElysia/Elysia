# How to Open the Elysia Interface

## 🚀 Quick Ways to Open

### Option 1: Double-Click Batch File (Easiest)
1. **Double-click**: `START_ELYSIA_INTERFACE.bat`
2. A new terminal window will open
3. The interface menu will appear
4. Type your choice (1-9) and press Enter

### Option 2: Use the New Batch File
1. **Double-click**: `OPEN_INTERFACE.bat` (just created)
2. Terminal opens with interface

### Option 3: Command Line
1. **Open Command Prompt** (Windows key + R, type `cmd`, Enter)
2. **Navigate to project**:
   ```cmd
   cd "C:\Users\mrnat\Project guardian"
   ```
3. **Run**:
   ```cmd
   python elysia_interface.py
   ```

---

## 📺 What You'll See

When the interface opens, you'll see:

```
======================================================================

  [4] Control System
  [5] View Logs
  [6] Introspection & Analysis
  [7] Open Web Dashboard
  [8] System Settings
  [9] Exit

======================================================================
Choice: _
```

---

## ⌨️ How to Use

1. **Type a number** (1-9) at the `Choice:` prompt
2. **Press Enter**
3. **Follow the prompts** in each section
4. **Press Enter** to return to the main menu

---

## 💡 Tips

- The terminal window is where you type
- Type numbers (1-9) to select options
- Press Enter after typing
- Type `back` or `exit` in chat mode to return to menu

---

## 🎯 Quick Start

**Right Now**: Double-click `START_ELYSIA_INTERFACE.bat` or `OPEN_INTERFACE.bat`

The terminal window will open and you can start using the interface!
