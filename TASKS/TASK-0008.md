# TASK-0008 — Bypass Detection Invariants (Network/File/Subprocess)

## Goal

Detect any external-action code paths that bypass TrustMatrix/IdentityAnchor gating.

## Scope

- tests/test_invariants.py
- scripts/* (optional)
- REPORTS/AGENT_REPORT.md
- CHANGELOG.md
- TASKS/TASK-0008.md
- CONTROL.md

## Non-goals

- No refactors of runtime code
- No new third-party dependencies

## Requirements

Add invariant checks that:

1. Scan repository source for network primitives:
   - requests, httpx, urllib, aiohttp, websocket, playwright network usage

2. Scan for filesystem writes:
   - open(..., "w"/"a"), Path.write_text/write_bytes, shutil.move/copy

3. Scan for subprocess execution:
   - subprocess.*, os.system

For each finding, assert one of:
- It occurs inside an approved gated module (WebReader or other explicit gateway)
- OR the enclosing function references TrustMatrix gate call
- OR the code is test-only

If the scan finds an ungated external action in non-test code, FAIL.

## Acceptance

- Invariant test reports findings and either PASS (all gated) or FAIL with exact file:line.
- acceptance.ps1 fails on detection of an ungated external action.

## Rollback

Revert invariant changes.
