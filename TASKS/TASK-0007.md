# TASK-0007 — Harden Trust/Mutation Contracts (No Ambiguity, No Hardcoded Policy)

## Goal

Make Trust denials explicit and auditable; remove ambiguous return values and hardcoded trust thresholds from mechanism layers.

## Scope (files allowed to touch)

- project_guardian/external.py
- project_guardian/mutation.py
- project_guardian/core.py (only if needed to pass new args)
- tests/test_invariants.py
- scripts/acceptance.ps1 (only if needed)
- REPORTS/AGENT_REPORT.md
- CHANGELOG.md
- TASKS/TASK-0007.md
- CONTROL.md

## Non-goals

- No broad refactor across unrelated modules
- No new dependencies beyond stdlib
- No redesign of TrustMatrix internals unless necessary for a gate method

## Requirements

1. WebReader.fetch denial handling:
   - Replace `return None` with either:
     A) raising `TrustDeniedError`, OR
     B) returning a structured result with `denied=True`
   - Update callers/tests accordingly.

2. Trust gating context:
   - Pass a context object/dict to TrustMatrix gate method including:
     - component
     - action
     - target (url/domain)
     - method (GET/POST if known)
     - caller identity if available
     - task id if available (or "unknown")
   - Ensure the context does not log sensitive content.

3. MutationEngine override:
   - Remove hardcoded `0.9+` threshold from MutationEngine.
   - Replace with a TrustMatrix decision call for "governance_mutation" (or "system_change") using context that includes:
     - list of touched paths
     - override flag state
     - caller identity (if available)

4. Tests:
   - Update tests to assert:
     - Denial is explicit (exception or structured result)
     - TrustMatrix received target context for WebReader
     - MutationEngine asks TrustMatrix for decision (not numeric threshold logic)

## Acceptance

- `.\scripts\acceptance.ps1` passes
- Invariant tests pass
- AGENT_REPORT includes:
  - example deny output (exception message or structured result)
  - evidence of context passed (e.g., via test spy/mock or logged field)

## Rollback

Revert changes to these modules and tests.
