# TASK-0005 — TrustGate WebReader.fetch()

## Goal

Ensure all network fetch operations are gated by TrustMatrix before execution.

## Scope (files allowed to touch)

- project_guardian/external.py (WebReader class)
- TrustMatrix / TrustEngine module ONLY if required to expose a stable gate method
- tests/test_invariants.py (only if needed to align with actual module names/paths)

## Non-goals

## Requirements

1. WebReader.fetch() must perform a TrustMatrix gate check BEFORE any request.

2. If gate denies:

   - Return a structured denial result or raise a specific exception (choose one and document)

3. Gate input must include at minimum:
   - action type = "network_fetch"
   - target URL / domain
   - caller/module identity if available (IdentityAnchor)

4. Logging: record allow/deny decision in a minimal way (no sensitive data dumps).
