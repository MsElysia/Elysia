# Archived Scripts

These scripts are kept for reference but are superseded by the main system:

- **start_simple_control_panel.py** - Minimal control panel (no GuardianCore). Use the integrated panel at http://127.0.0.1:5000 when running elysia.py.
- **quick_dashboard.py** - Standalone dashboard. Use the integrated UIControlPanel instead.
- **CONTROL_PANEL_VERIFICATION_REPORT.md** / **CONTROL_PANEL_COMPLETE_VERIFICATION.md** - Verification notes from past debugging.
