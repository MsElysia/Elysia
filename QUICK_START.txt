ELYSIA - QUICK START GUIDE
================================================================================

HOW TO USE ELYSIA IN 3 STEPS:

STEP 1: START THE SYSTEM
   Double-click: INTERACT_ELYSIA.bat

STEP 2: USE THE MENU
   Choose from:
   1. Store Memory - Save information
   2. Recall Memories - View what you saved
   3. Show System Status - Check system health
   4. Show Startup Verification - See components
   5. Search Memories - Find by category
   6. Exit - Shut down

STEP 3: START USING IT!
   Example:
   - Store: "Buy groceries" (category: tasks)
   - Store: "Learned Python today" (category: learning)
   - Recall: See your recent memories
   - Search: Find all "tasks" memories

================================================================================
WHAT IS ELYSIA?
================================================================================

Elysia is a memory system that:
- Remembers everything you tell it
- Organizes information by category
- Lets you search and recall information
- Persists data between sessions
- Helps you build a knowledge base

================================================================================
COMMON CATEGORIES
================================================================================

Use these categories to organize:
- tasks      → Things to do
- learning   → Things you learned
- ideas      → Ideas and thoughts
- project    → Project-related info
- notes      → General notes
- meetings   → Meeting notes
- code       → Code snippets/notes
