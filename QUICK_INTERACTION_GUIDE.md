# Quick Interaction Guide - Elysia

## 🚀 Fastest Way to See What Elysia Is Doing

### Option 1: Web Dashboard (Best Overview) ⭐
```cmd
python start_ui_panel.py
```
Then open: `http://127.0.0.1:5000`

**See**:
- Real-time system status
- What Elysia is thinking about
- Active tasks
- Memory statistics
- Recent activity

---

### Option 2: Chat Interface (Easiest)
```cmd
CHAT_ELYSIA.bat
```
**Ask**:
- "What have you been learning?"
- "What are you thinking about?"
- "Show me your status"

---

### Option 3: Status Check (Quick)
```cmd
python show_status.py
```
Shows current component status
