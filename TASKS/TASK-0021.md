# TASK-0021 — CONTROL.md Integration + Minimal Task Router (No Auto-Execution)

## Goal

Close the loop between the control panel and Core by making GuardianCore read `CONTROL.md` and deterministically route the current task **without executing it yet**. Also remove ambiguous string error returns from the core loop path by returning structured results.

## Scope (allowed files)

* `project_guardian/core.py`
* `CONTROL.md` (read-only semantics; may add comments but do not change format)
* `tests/test_core_task_router.py` (new)
* `SPEC_MODULES/core.md` (update)
* `REPORTS/module_audit_core.md` (update)
* `TASKS/TASK-0021.md` (new)
* `CHANGELOG.md`
* `REPORTS/AGENT_REPORT.md`
* `CONTROL.md` (set CURRENT_TASK back to NONE at end)

## Non-goals

* Do NOT execute tasks automatically
* Do NOT add new dependencies
* Do NOT touch UI code
* Do NOT refactor unrelated code

## Required behavior

### 1) CONTROL.md parsing

Add a helper in Core (or module-level) to read CONTROL.md:
* Parse `CURRENT_TASK: <value>`
* Return the value normalized:
  * `"NONE"` → `None`
  * `"TASK-XXXX"` → `"TASK-XXXX"`
* Handle missing file or missing key safely:
  * Return structured error result (see below)

### 2) Task contract loading

Add `load_task_contract(task_id)`:
* Reads `TASKS/{task_id}.md`
* If file missing → return `{status:"error", code:"TASK_NOT_FOUND", task_id, detail}`
* If file exists → return `{status:"ready", task_id, contract_hash, contract_preview}`
  * `contract_hash`: sha256 of full file contents
  * `contract_preview`: first ~20 lines (for visibility), safe/plaintext

### 3) Integrate router into `run_once()`

Update `GuardianCore.run_once()` so it returns structured results:
* If CURRENT_TASK is None:
  * `{status:"idle", current_task: None}`
* If CURRENT_TASK exists and task loads:
  * `{status:"ready", current_task: task_id, contract_hash, contract_preview}`
* If errors occur:
  * `{status:"error", code, detail, current_task}`

**Critical:** No "return string message" paths from run_once. Only dict results.

### 4) Backward compatibility note

If other existing methods return strings today, leave them for now. Only enforce structured results for the **run_once() path**.
