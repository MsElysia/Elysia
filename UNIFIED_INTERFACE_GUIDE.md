# Elysia Unified Interface Guide

## 🚀 Quick Start

**Start the Unified Interface**:
- Double-click: `START_ELYSIA_INTERFACE.bat`
- Or run: `python elysia_interface.py`

---

## 📋 Main Menu Options

### [1] 📊 View System Status
- Memory statistics
- Component status
- System uptime
- Active tasks

### [2] 💬 Chat with Elysia
- Natural conversation
- Ask questions
- Get help
- Commands:
  - `status` - Show system status
  - `memories` - View memories
  - `what are you thinking` - See focus
  - `back` - Return to menu

### [3] 🧠 View Memories & Learning
- Total memories
- Recent memories
- Search memories
- Learning statistics

### [4] 🎮 Control System
- Pause/Resume system
- Create snapshots
- Trigger introspection
- System controls

### [5] 📝 View Logs
- Main log (elysia_unified.log)
- Unified log (full history)
- View last 50 lines
- Open log files

### [6] 🔍 Introspection & Analysis
- What Elysia is thinking
- Behavior patterns
- Memory health
- Focus analysis

### [7] 🌐 Open Web Dashboard
- Starts web UI automatically
- Opens browser
- Full-featured dashboard
- Real-time updates

### [8] ⚙️ System Settings
- View configuration
- Check dependencies
- Test components
- System info

### [9] ❌ Exit
- Clean shutdown
- Save state

## 🎯 Usage Examples

### Check System Status
1. Start interface
2. Choose `[1]`
3. View status
4. Press Enter to continue

### Chat with Elysia
1. Choose `[2]`
2. Type your message
3. Get response
4. Type `back` to return

### Open Web Dashboard
1. Choose `[7]`
2. Wait for server to start
3. Browser opens automatically
4. Access full dashboard
