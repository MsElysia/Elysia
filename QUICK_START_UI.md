# Quick Start - UI

## To Open the UI

**Double-click**: `START_ELYSIA_INTERFACE.bat`

Or run:
```cmd
python elysia_interface.py
```

## What Happens

1. **Autonomous system starts** (learning, internet scanning, all modules)
2. **Interface menu appears**
3. **Both run together**

## Menu Options

Type a number (1-9) and press Enter:

- **[1]** View System Status
- **[2]** Chat with Elysia  
- **[3]** View Memories & Learning
- **[4]** Control System
- **[5]** View Logs
- **[6]** Introspection & Analysis
- **[7]** Open Web Dashboard (full web UI)
- **[8]** System Settings
- **[9]** Exit

## Quick Access

**Right now**: The interface should be starting. Check your terminal window!

If you don't see it, double-click `START_ELYSIA_INTERFACE.bat` in File Explorer.
