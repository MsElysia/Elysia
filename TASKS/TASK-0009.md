# TASK-0009 — Trust Decision Object + Consistent Denial Handling

## Goal

Make TrustMatrix return a structured decision object and standardize how modules respond.

## Scope

- project_guardian/trust.py
- project_guardian/external.py
- project_guardian/mutation.py
- tests/test_invariants.py
- REPORTS/AGENT_REPORT.md
- CHANGELOG.md
- TASKS/TASK-0009.md
- CONTROL.md

## Requirements

1. Add a TrustDecision object (dataclass ok) with:
   - allowed: bool
   - decision: "allow"|"deny"|"review"
   - reason_code: str
   - message: str (human readable)
   - risk_score: float (0..1) optional

2. validate_trust_for_action returns TrustDecision (not bare bool).

3. WebReader / MutationEngine:
   - raise TrustDeniedError on deny with reason_code + context
   - on "review" raise a distinct exception or return a structured "needs_review" response (choose one)

## Acceptance

- Tests updated accordingly and pass
- No caller interprets raw trust floats
